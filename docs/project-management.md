# 📝 User Stories

## Authentication

### Story

As a user,

I want to register an account,

so that I can save my data.

---

Acceptance Criteria:

* Email can be entered.
* Password validation exists.
* Account is created.
* User is redirected to dashboard.

Story Points:

3

---

## Planner

### Story

As a user,

I want to create tasks,

so that I can track work.

---

Acceptance Criteria:

* Task title required.
* Task saved successfully.
* Task visible immediately.

Story Points:

2

---

## AI Chat

### Story

As a user,

I want to chat with Cheta,

so that I can receive guidance.

---

Acceptance Criteria:

* Message sent.
* Response returned.
* Conversation saved.

Story Points:

5

---

## Memory

### Story

As a user,

I want Cheta to remember important things,

so that future conversations are personalised.

---

Acceptance Criteria:

* Memory stored.
* Memory retrieved.
* User can delete memory.

Story Points:

8

* Configure TypeScript
* Configure Tailwind
