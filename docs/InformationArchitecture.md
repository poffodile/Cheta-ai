# Cheta

> **Remember. Plan. Grow.**

---

# 🏗️ Information Architecture and Page Structure

## **Overview**

Cheta is designed as a personal operating system.

The navigation and layout are inspired by:

* Notion
* Linear
* Arc Browser
* Apple

Cheta aims to provide a clean and calm environment while allowing users to access planning, memory, projects, goals, and AI assistance from anywhere.

---

---

# 🔍 Global Search

Search should be available throughout the application.

Users should be able to search:

* Tasks
* Memories
* Conversations
* Projects
* Notes

Search bar location:

* Top navigation bar
* Accessible from every page

---

# 🤖 AI Availability

Chat should not be limited to one page.

A floating AI button should be available globally.

Users should be able to ask questions from anywhere.

Examples:

> What should I focus on this week?

> Have I forgotten anything?

> Summarise my goals.

> Turn this idea into a project.

---

### Progressive Disclosure

Advanced features should appear gradually.

### Consistency
