# Cheta

> Remember. Plan. Grow.

# 🎨 Design System

## Overview

Cheta follows a calm, premium, and modern design philosophy inspired by:

* Apple
* Arc Browser
* Notion
* Linear

The interface should feel:

* Warm
* Professional
* Minimal
* Calm
* Human

Cheta should avoid unnecessary clutter and allow users to focus on progress and reflection.
