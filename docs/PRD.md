Version 1.0

⸻

Product Overview

Cheta is an AI-powered personal companion designed to help users remember what matters, plan effectively, and turn intentions into actions.

Cheta combines planning, memory, projects, goals, and AI guidance into a single platform that grows alongside its users.

⸻

Product Goals

The goal of Cheta is to:

* Help users stay organised.
* Reduce forgotten tasks and ideas.
* Provide meaningful AI guidance.
* Transform goals into actionable plans.
* Encourage consistent personal growth.
* Create one place for projects, planning, career development, and reflection.

Functional Requirements

The system shall allow users to:

Authentication

* Register accounts.
* Sign in.
* Reset passwords.

Planner

* Create tasks.
* Update tasks.
* Delete tasks.
* Mark tasks complete.
* Filter tasks.

AI Chat

* Send messages.
* Receive responses.
* Save conversations.
* Generate suggestions.

Memory

* Save important information.
* Retrieve memories.
* Edit memories.
* Delete memories.

Dashboard

* Display task progress.
* Display AI suggestions.
* Display recent activity.

⸻

Non-Functional Requirements

Performance

* Dashboard should load quickly.
* Chat responses should feel responsive.

Security

* User data should be protected.
* Users should only access their own data.

Reliability

* Conversations should not be lost.
* Tasks should persist correctly.

Scalability

* System should support growth.
* Architecture should remain modular.

Usability

* Simple interface.
* Easy navigation.
* Mobile-friendly layouts.

⸻

Success Criteria

Version 0.1 is successful if users can:

* Create and manage tasks.
* Chat with Cheta.
* Receive useful suggestions.
* Save and retrieve memories.
* See progress through the dashboard.
